CSS pseudo-class:

  1. :root : it is ability to reach the element in tag HTML
  2. :hover: when u not reach the element, it has another adjustment
  3. :active: when u not click the element, it has another adjustment
  4. :first-child: the first element
  5. :last-child: the last element

CSS pseudo-element:

  • ::before: meaning the first element, it always appear the first
  • ::after: meaning the last element, it always appear the last
  • ::first-letter: meaning chang the attribute first character
  • ::first-line: meaning change the attribute first line
  • ::selection: when u paint block, it change the attribute ::)
